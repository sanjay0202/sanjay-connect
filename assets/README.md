# Assets Folder

This folder contains static assets for the portfolio website.

## Structure

- **images/** - Store images, logos, and graphics here
- **documents/** - Store downloadable documents here

## Usage

### Images
Place any images you want to use in the website in the `images/` folder.

Example:
- Profile photo
- Project screenshots
- Company logos
- Certification badges

### Documents
Place downloadable documents in the `documents/` folder.

Example:
- Resume PDF (currently in `my-page/sanjayResume-26.pdf`)
- Certificates
- Case studies
- White papers

## Notes

- The resume is currently located at `my-page/sanjayResume-26.pdf`
- All external resources (icons, fonts) are loaded via CDN
- Images can be added and referenced in HTML as needed
